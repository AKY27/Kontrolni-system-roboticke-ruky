## 3. výstup (1. 2. 2026)

Ve třetím výstupu jsem přepsal program pro ovládání mikrokontroleru do jazyka C++, také jsem se rozhodl zjednodušit
některé jeho funkce, které jsem udělal v minulém výstupu a přesunout je do python knihovny, tak abych se 
přesněji držel specifikace projektu. Kód stále využívá stejné systémy jako např.: PWM pro ovládání 
servo motorů, ale je napsaný v jazyku C++, pro kompilaci využívám kompilátor CMake a pro kompatibilitu
s Raspberry Pi Pico využívám oficiální software development kit. Do python knihovny jsem implementoval
některé funkce které byly dříve na mikrokontroleru. Začal jsem také psát demostrační kód pro knihovnu.
Ruka jde niní ovládat ovladačem dualsense. Všechny projektové soubory jsou opět dostupné na Git Hubu, odkaz na daný
repositář je v příloze.
