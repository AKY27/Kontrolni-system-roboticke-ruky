## 4. výstup (16. 3. 2026)

Hlavním cílem čtvrtého výstupu bylo vytvořit webové rozhraní pro ovládání robotické ruky. Rozhraní jsem napsal 
v čistém HTML, CSS a JavaScriptu. Pro stylování jsem využil framework Tailwind CSS a pro 3D vizualizaci 
knihovnu Three.js, pomocí které jsem sestavil interaktivní model robotické ruky. K programování webového rozhraní
jsem využil pomoc AI, protože se v tomto oboru osobně nepohybuji. Rozhraní také zobrazuje světové souřadnice a úhel 
náklonu chapadla vypočítané z kinematiky modelu. Je nyní možno ruko ovládat také přes světové souřadnice, ale
tento systém ještě není dopracovaný a dělá chyby. Na straně serveru jsem využil Python framework Flask, který 
poskytuje REST API s koncovými body pro čtení a nastavení úhlů servomotorů. Webový server je integrován přímo 
do Python knihovny z předchozích výstupů. Ruku je nyní možné ovládat z libovolného zařízení v lokální síti 
přes webový prohlížeč pomocí posuvníků pro každý kloub. Všechny projektové soubory jsou opět dostupné na 
Git Hubu, odkaz na daný repositář je v příloze.

---

server jde zapnout pomocí příkazů:
```
cd ./code/API
./venv/bin/python -m RoborukaAPI
```
